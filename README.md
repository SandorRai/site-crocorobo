# CROCOROBO Robotics Team Project

## Project Overview
This project involves the development of a robot designed to help protect artifacts found in the golden environment and promote them. The project will also include an Augmented Reality (AR) experience featuring Dacian ovens and an interactive game.

## Design Elements
- **Color Scheme**: Shades of green, beige, and purple to create a playful and child-friendly atmosphere.

## Features
1. **Project Presentation**: A detailed description of the innovation project.
2. **AR Integration**: Interactive AR experience showcasing Dacian ovens.
3. **Interactive Game**: A fun game related to the project theme.

## Technologies Used
- HTML, CSS, JavaScript for the front-end.
- AR libraries for the augmented reality features.

## Future Enhancements
- Additional interactive elements.
- Community engagement features.
